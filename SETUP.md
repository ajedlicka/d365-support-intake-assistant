# Optional Setup — Save It So You Don't Paste It Every Time

**You don't need this.** Pasting the contents of [INTAKE-PROMPT.md](INTAKE-PROMPT.md) into a
fresh chat works exactly the same, every time. This page is only for people who use the tool
often enough that pasting gets tedious.

Most AI assistants let you save a set of instructions once, so every new chat already knows to
run the intake. Here's how, for the four common tools. Pick the one you use — you don't need
the others.

Throughout, "the instructions" means the full contents of **INTAKE-PROMPT.md**. Open that
file, select all, and copy it before you start.

---

## Claude

*Requires a paid plan (Pro or Team). Not available on the Free plan.*

1. In the left sidebar, click **Projects**, then **Create Project**. Give it a name like
   "D365 Ticket Intake."
2. Open the project and find **Project instructions** (also called "Set project
   instructions").
3. Paste the full instructions there and save.
4. From now on, start a **new chat inside that project** and just describe your issue — the
   instructions are already loaded. You don't paste anything.

---

## ChatGPT

*Requires a paid plan (Plus or above). Two options — a Project is simpler, a Custom GPT is
more durable.*

**Option A — Project (simpler):**

1. In the left sidebar, click **+ New project** (or the projects icon) and name it.
2. Open the project's **Instructions** (sometimes under the project's settings / "Add
   instructions").
3. Paste the full instructions and save.
4. Start any chat inside that project and describe your issue.

**Option B — Custom GPT (more durable — gets its own entry in your sidebar):**

1. Click **Explore GPTs**, then **+ Create** (top right).
2. Go to the **Configure** tab.
3. Paste the full instructions into the **Instructions** box. Give it a name and, optionally,
   a short description.
4. Save. It now lives in your sidebar; open it and describe your issue whenever you need it.

---

## Gemini

*Gem creation is available on the free tier.*

1. In the left panel, find **Gems** and click **New Gem** (or **Gem manager → New Gem**).
2. Give it a name like "D365 Ticket Intake."
3. Paste the full instructions into the **Instructions** field and save.
4. Select that Gem whenever you want to run an intake, then describe your issue.

---

## Copilot

*What's possible depends on your Copilot license — check with your IT team.*

- **Microsoft 365 Copilot (licensed):** A saved agent may be possible, but setting one up
  usually needs IT involvement (and sometimes admin approval). Ask your IT team whether they
  can publish this as an agent for your team.
- **Copilot Chat (free / built into Edge or Windows):** There's no saved-agent equivalent.
  Paste the instructions at the start of each conversation, the same way you would in any
  chat.

If you're not sure which one you have, ask your IT team — and mention you'd like to reuse a
fixed set of instructions, so they can point you to the right option.

---

## Two things to remember

1. **Plans and features change often.** The menus and plan requirements above are accurate at
   the time of writing, but these tools update frequently. If something's moved, check the
   tool's current help documentation — the idea ("save a set of instructions") is the same
   even when the buttons move.

2. **Re-copy the instructions now and then.** This tool gets updated over time. If you've
   saved a copy into a Project, GPT, or Gem, replace it with a fresh copy of
   INTAKE-PROMPT.md every so often so you're running the current version.
